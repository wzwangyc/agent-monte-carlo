## 一、核心研究问题

### 1.1 科学问题

**主问题：**
> 基于异质 Agent 相互作用的模拟模型，能否重现真实金融市场的典型事实（stylized facts）？

**子问题：**
1. 需要哪些 Agent 类型？（异质性来源）
2. Agent 如何决策？（微观行为规则）
3. Agent 如何学习？（适应性机制）
4. 市场如何出清？（价格形成机制）
5. 如何验证模型"成功"？（检验标准）

---

### 1.2 研究贡献

**理论贡献：**
1. 整合多个文献中的 Agent 类型到统一框架
2. 使用 EWA 学习（有实验支持）而非 ad hoc 规则
3. 使用 Glosten-Milgrom 做市商（有微观基础）
4. 提供完整的校准和验证流程

**实证贡献：**
1. 校准到 50 年 S&P 500 数据
2. 验证 5+ 个典型事实
3. 进行敏感性分析和稳健性检验
4. 开源代码和数据

**方法贡献：**
1. 参数先验来自文献，后验来自校准
2. 样本内校准 + 样本外验证
3. 统计检验而非目视对比
4. 可复现的研究流程

## 二、理论假设

### 2.1 核心假设

**H1: 市场是复杂适应系统 (CAS)**
- 由异质 Agent 组成
- Agent 有限理性
- Agent 之间互动
- 宏观现象从微观互动中涌现

**可检验推论：**
- 推论 1.1: 单一 Agent 类型无法重现典型事实
- 推论 1.2: 增加 Agent 异质性提高模型拟合度
- 推论 1.3: 学习机制改善长期拟合

---

**H2: 行为偏差是系统性的**
- 不是随机噪声
- 可量化建模
- 参数可从实证估计

**可检验推论：**
- 推论 2.1: 包含行为偏差的模型优于理性预期模型
- 推论 2.2: 行为参数校准值接近实验估计值
- 推论 2.3: 移除行为偏差后模型拟合恶化

---

**H3: 学习是适应性的**
- Agent 从经验中学习
- 学习速度有限
- 学习环境非平稳

**可检验推论：**
- 推论 3.1: 有学习的模型优于无学习模型
- 推论 3.2: EWA 学习优于标准 Q-Learning
- 推论 3.3: 学习参数在实证范围内

---

**H4: 价格由订单流驱动**
- 订单流包含信息
- 价格冲击持久
- 流动性有限

**可检验推论：**
- 推论 4.1: 订单流可预测价格变化
- 推论 4.2: Kyle's lambda 在实证范围内
- 推论 4.3: 价差与信息不对称正相关

---

### 2.2 辅助假设

**A1: 市场出清机制**
- 做市商提供流动性
- 价差内生决定
- 库存风险管理

**A2: Agent 类型稳定性**
- Agent 类型固定
- 但策略权重可演化
- 类型比例可校准

**A3: 时间离散性**
- 日度时间步长
- 符合数据频率
- 简化计算

## 四、模型形式化

### 4.1 经济环境

**时间：** 离散，t = 0, 1, ..., T

**资产：** 单一风险资产，供给固定为 1

**价格：** P_t，由市场决定

**基本面价值：** V_t，遵循随机过程
```
V_t = V_{t-1} · exp(μ + σ_V · ε_t)
ε_t ~ N(0, 1)
```

---

### 4.2 Agent 设定

**Agent 总数：** N（外生，可变化进行敏感性分析）

**Agent 类型：** K 种（K 可变化）

**类型 i 的比例：** w_i，满足 Σw_i = 1

**Agent 状态：**
```
State_{i,t} = {
    cash: 现金持有,
    holdings: 资产持有,
    wealth: 总财富,
    beliefs: 信念状态,
    memory: 记忆状态
}
```

---

### 4.3 Agent 决策

**通用决策框架：**
```
1. 观察：O_{i,t} = (P_t, V_t, 其他信息)
   ↓
2. 感知：S_{i,t} = Perception(O_{i,t})
   ↓
3. 决策：A_{i,t} = Decision(S_{i,t}, θ_i)
   ↓
4. 执行：提交订单 (quantity, direction)
   ↓
5. 学习：θ_{i,t+1} = Learning(θ_{i,t}, reward)
```

**具体 Agent 类型：**

#### (1) Noise Trader（噪声交易者）

**行为规则：**
```
P(buy) = 0.5 + η_t
η_t ~ N(0, σ_noise)

quantity ~ Poisson(λ_q)
```

**文献：** Black (1986), French & Roll (1986)

**参数：**
- σ_noise ∈ [0.1, 0.3]（噪声强度）
- λ_q ∈ [1, 10]（平均交易量）

---

#### (2) Momentum Trader（动量交易者）

**行为规则：**
```
# 基于 Hvidkjaer (2006)
signal = α + β_12m1 · J12M1 + β_6m1 · J6M1

J12M1 = (P_{t-21} - P_{t-273}) / P_{t-273}  # 12 个月动量，跳过 1 个月
J6M1 = (P_{t-21} - P_{t-147}) / P_{t-147}   # 6 个月动量

P(buy) = 1 / (1 + exp(-signal))
```

**文献：** 
- Daniel et al. (1998) - 过度自信与动量
- Hvidkjaer (2006) - 参数估计

**参数：**
- α ∈ [-1.0, 0]（截距）
- β_12m1 ∈ [1.0, 2.0]（长期动量）
- β_6m1 ∈ [0.5, 1.5]（短期动量）
- overconfidence ∈ [1.0, 2.0]（过度自信放大）

---

#### (3) Value Trader（价值交易者）

**行为规则：**
```
# 基于 De Bondt & Thaler (1985)
mispricing = (V_t - P_t) / P_t

if mispricing > threshold_buy:
    P(buy) = high
elif mispricing < threshold_sell:
    P(sell) = high
else:
    P(hold) = high
```

**文献：**
- De Bondt & Thaler (1985) - 均值回归
- Lakonishok et al. (1994) - 价值策略

**参数：**
- threshold_buy ∈ [0.05, 0.2]（买入阈值）
- threshold_sell ∈ [-0.2, -0.05]（卖出阈值）
- mean_reversion_speed ∈ [0.05, 0.2]（回归速度）

---

#### (4) Herd Trader（羊群交易者）

**行为规则：**
```
# 基于 Shiller (1995)
net_buy = mean(others_actions[-window:])

P(follow) = 1 / (1 + exp(-(γ_0 + γ_1 · net_buy)))

if random() < P(follow):
    action = majority_action
else:
    action = random_choice
```

**文献：**
- Shiller (1995) - 羊群行为综述
- Scharfstein & Stein (1990) - 声誉驱动羊群

**参数：**
- γ_0 ∈ [-0.5, 0.5]（基准倾向）
- γ_1 ∈ [0.3, 1.0]（羊群强度）
- window ∈ [5, 20]（观察窗口）

---

### 4.4 学习机制（EWA）

**通用 EWA 框架（Camerer & Ho, 1999）：**

**吸引度更新：**
```
A_{i,s}(t) = [φ·N(t-1)·A_{i,s}(t-1) + (δ + (1-δ)·I(s, s*)) · π_{i,s}(t)] / N(t)

经验权重更新：
N(t) = ρ·N(t-1) + 1

策略选择（Logit）：
P(s) = exp(λ·A_{i,s}(t)) / Σ_{s'} exp(λ·A_{i,s'}(t))
```

**参数：**
- φ ∈ [0.7, 1.0]（吸引度衰减）
- δ ∈ [0.3, 0.8]（想象权重）
- ρ ∈ [0.7, 1.0]（经验增长）
- λ ∈ [0.5, 3.0]（选择敏感度）

**来源：** Camerer & Ho (1999) 表 III，12 个实验的估计

---

### 4.5 市场机制

**做市商定价（Glosten & Milgrom, 1985）：**

**价差决定：**
```
spread_t = 2 · μ · σ_V + 2c + κ · |inventory_t|

bid_t = V_t · (1 - spread_t/2)
ask_t = V_t · (1 + spread_t/2)
```

**参数：**
- μ ∈ [0.05, 0.3]（知情交易者比例）
- σ_V ∈ [0.01, 0.1]（价值波动）
- c ∈ [0.0001, 0.001]（交易成本）
- κ ∈ [0.001, 0.01]（库存风险）

---

**价格冲击（Kyle, 1985）：**

**订单流影响：**
```
ΔP_t = λ · Y_t + ε_t

Y_t = 净订单流（买为正，卖为负）
λ = Kyle's lambda ∈ [0.01, 0.05]
```

**来源：** Hasbrouck (1991) 实证估计

## 五、校准与验证框架

### 5.1 校准目标

**目标矩向量：**
```
m_data = [
    kurtosis,           # 峰度 ≈ 19
    acf1_squared,       # 波动率 ACF(1) ≈ 0.2
    crash_frequency,    # 崩盘频率 ≈ 0.03/年
    vol_volume_corr,    # 量 - 波相关 ≈ 0.6
    momentum_profit     # 动量收益 ≈ 0.5%/月
]
```

**来源：** Cont (2001), Hvidkjaer (2006), 本文计算

---

### 5.2 距离函数

**加权距离：**
```
D(θ) = Σ_{j=1}^J w_j · [(m_sim,j(θ) - m_data,j) / m_data,j]²

权重：
w = [1.0, 1.0, 1.0, 0.5, 0.5]  # P0 权重 1.0, P1 权重 0.5
```

**优化：**
```
θ* = argmin_θ D(θ)

方法：Nelder-Mead 或 CMA-ES
初始值：参数先验均值
```

---

### 5.3 验证假设

**统计检验框架：**

| 假设 | 检验方法 | 原假设 | 通过标准 |

**通过标准：**

## 六、可证伪性

### 6.1 什么结果会证伪模型？

**强证伪（模型失败）：**
1. 峰度 < 10 或 > 30（无法重现肥尾）
2. ACF(r²) 不显著（p > 0.05）
3. 崩盘频率 < 1% 或 > 10%
4. KS 检验拒绝（p < 0.01）

**弱证伪（需要调整）：**
1. 某些目标矩匹配不佳（差异 > 30%）
2. 样本外表现显著恶化
3. 参数校准值远离先验

---

### 6.2 什么结果会支持模型？

**强支持：**
1. 所有 P0 假设通过统计检验
2. 参数校准值在先验范围内
3. 样本外预测准确
4. 稳健性检验全部通过

**中等支持：**
1. 大部分 P0 假设通过
2. 定性特征匹配（即使定量有偏差）
3. 反事实预测合理

## 七、与现有模型的关系

### 7.1 与传统 MC 对比

| 维度 | 传统 MC | Agent MC |
|------|---------|----------|
| **收益率分布** | 正态 | 肥尾（涌现） |

| 维度 | 纯 ABM | Agent MC |

| **校准** | 少 | 全面 |
