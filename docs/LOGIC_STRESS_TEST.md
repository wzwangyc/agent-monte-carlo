## 一、逻辑一致性测试

### 1.1 核心假设的自洽性

#### 测试 1: H1 与 H3 是否矛盾？

**H1:** 市场是复杂适应系统（CAS）
- Agent 异质
- 有限理性
- 适应性学习

**H3:** 学习是适应性的
- Agent 从经验中学习
- 学习速度有限
- 环境非平稳

**潜在矛盾：**
```
质疑：如果环境非平稳（H3），Agent 如何学习？
      学习需要稳定性，但 CAS 本质不稳定
```

**分析：**
```
反驳 1（弱）：Agent 学习局部规律
  - 问题：局部规律存在吗？CAS 中可能没有局部稳定性

反驳 2（强）：Agent 学习的是"适应性策略"
  - 不是学习"世界模型"
  - 而是学习"在不确定中生存的策略"
  - 类似进化：不是预测环境，而是适应环境

反驳 3（最强）：学习本身改变环境（反身性）
  - Agent 学习 → 策略改变 → 环境改变 → 学习更新
  - 这是 co-evolution，不是单向学习
  - 与 H1 一致（CAS 的核心特征）
```

**结论：** ⚠️ **需要澄清**
- 当前框架中的"学习"概念模糊
- 应明确为"适应性策略演化"而非"预测学习"
- 建议修改 H3 表述

**修改建议：**
```
原 H3: 学习是适应性的
新 H3: Agent 策略通过演化机制适应环境
      (evolutionary adaptation, not predictive learning)

关键区别：
- 不是"学习预测价格"
- 而是"学习在市场中生存"
- 类比生物进化，而非机器学习
```

```
但框架中是否明确建模了这些？
```

**检查当前框架：**

#### 测试 4: 学习机制与决策规则是否一致？

**当前设定：**
- 决策规则：基于文献的特定公式（如 Momentum 基于 Hvidkjaer）
- 学习机制：EWA（更新策略权重）

**质疑：**
```
1. 如果 Agent 用 EWA 学习，
   为什么决策规则是固定的？

2. EWA 学习的是什么？
   - 策略权重？但策略只有 1 个（Momentum）
   - 参数？但参数来自文献，不应学习

3. 如果参数可学习，
   为什么还要文献先验？
```

**分析：**
```
当前框架的混淆：
- 说用 EWA 学习
- 但决策规则是固定的（Momentum 公式）
- 参数是先验的（来自文献）

这自相矛盾！

如果 EWA 学习策略权重：
- 需要多个策略供选择
- 但每个 Agent 只有 1 个策略

如果 EWA 学习参数：
- 参数应该收敛
- 但文献说参数是先验的
```

**结论：** ❌ **严重逻辑矛盾**

**修复方案（3 选 1）：**

**方案 A：纯 EWA 学习（推荐）**
```python
class LearningAgent(Agent):
    """
    Agent 不预设类型，通过 EWA 学习策略
    
    策略空间：
    - Momentum
    - Value
    - Herding
    - Random
    
    EWA 学习选择哪个策略
    """
    def __init__(self):
        self.ewa = EWALearning(n_strategies=4)
        self.strategies = [Momentum(), Value(), Herding(), Random()]
    
    def decide(self):
        # EWA 选择策略
        strategy_idx = self.ewa.choose_strategy()
        strategy = self.strategies[strategy_idx]
        return strategy.decide(observation)
    
    def learn(self, reward):
        # 更新所有策略的吸引度
        foregone = [s.estimate_reward() for s in self.strategies]
        self.ewa.update(chosen, reward, foregone)
```

**方案 B：纯规则型（放弃 EWA）**
```python
class MomentumAgent(Agent):
    """
    Agent 类型固定，参数固定
    
    不学习，只是执行规则
    """
    def __init__(self):
        # 参数来自文献，不学习
        self.beta = 1.5  # Hvidkjaer (2006)
    
    def decide(self):
        # 固定规则
        return momentum_decision()
```

**方案 C：混合（复杂但真实）**
```python
class HybridAgent(Agent):
    """
    核心规则固定（来自文献）
    但参数可微调（学习）
    
    类似：先天 + 后天
    """
    def __init__(self):
        self.base_params = get_from_literature()  # 先验
        self.adjustment = 0.0  # 可学习
    
    def learn(self, reward):
        # 微调参数，但有约束
        self.adjustment += learning_rate * reward
        self.adjustment = clip(self.adjustment, -0.5, 0.5)  # 不偏离先验太多
```

**推荐：方案 A**
- 逻辑一致
- 有文献支持（Kirman, 1993）
- 更灵活

### 1.3 校准与验证的逻辑

#### 测试 5: 校准目标是否独立？

**当前目标矩：**
1. 峰度 ≈ 19
2. ACF(1) ≈ 0.2
3. 崩盘频率 ≈ 0.03/年
4. 量 - 波相关 ≈ 0.6
5. 动量收益 ≈ 0.5%/月

**质疑：**
```
1. 这些目标矩是否独立？
   - 峰度和崩盘频率可能相关
   - ACF 和量 - 波相关可能相关

2. 如果目标矩相关，
   加权距离函数是否合理？

3. 5 个目标矩 vs 10+ 个参数
   是否欠识别？
```

**分析：**
```
相关性检查：
- 峰度 vs 崩盘频率：高相关（都捕捉尾部）
- ACF vs 量 - 波相关：低相关
- 动量收益 vs 其他：低相关

问题：
- 目标矩数量 (5) < 参数数量 (10+)
- 欠识别：多组参数都能匹配目标

后果：
- 校准结果不唯一
- 参数解释力弱
- 审稿人会质疑
```

**结论：** ⚠️ **欠识别问题**

#### 测试 9: 与有效市场假说 (EMH) 的关系？

**质疑：**
```
1. 如果模型成功重现市场事实，
   是否支持或反驳 EMH？

2. 模型中有理性 Agent 吗？
   - 如果没有，是否预设 EMH 错误？
   - 如果有，为什么他们不消除偏差？

3. 审稿人来自 EMH 学派怎么办？
```

**分析：**
```
当前框架的立场：
- 预设行为偏差存在（H2）
- 预设有限理性
- 这与 EMH 矛盾

但需要明确：
- 不是"EMH 完全错误"
- 而是"EMH 在某些条件下失效"

更好的框架：
- EMH 是基准（null hypothesis）
- 行为模型是替代（alternative）
- 检验哪个更符合数据
```

**结论：** ⚠️ **需要澄清与 EMH 的关系**

**修复方案：**
```
在框架中添加：

"本文不是要'反驳'EMH，而是：
1. 检验在什么条件下 EMH 成立/失效
2. 行为偏差是否显著改善拟合
3. 有限理性的边界在哪里

方法：
- 基准模型：纯理性 Agent（EMH）
```
